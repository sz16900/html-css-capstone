### Prerequisites

- Node.js (for Stylelint)
- Web Browser
- CLI
- IDE

### Dependencies

- Stylelint

### Install

`git clone`

`npm install`

### Run Linter Test

`stylelint "*/*.css"`

> https://stylelint.io/user-guide/cli

### Autofix CSS Linting Errors

`stylelint "*/**.css" --fix`

### Deployment

`npm start`
